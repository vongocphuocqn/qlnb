Tải file dữ liệu từ phần mềm HIS

Mỗi lần tải là một khung thời gian (đợt). Tất cả các đợt được lưu lại và cộng dồn tự động khi tính bảng tổng hợp theo tháng/khoảng ngày. Định dạng nhận: .xlsx, .xls, .csv. Nếu chỉ có file PDF từ HIS, hãy xuất lại dưới dạng Excel/CSV trong phần mềm HIS để bảo đảm cột dữ liệu không bị lẫn.

📄 Kéo–thả file vào đây, hoặc bấm để chọn file
(có thể chọn nhiều file cùng lúc — mỗi file 1 đợt riêng)
Ánh xạ cột dữ liệu —

Chỉ định mỗi cột trong file tương ứng với vai trò nào. Bấm "Lưu cấu hình ánh xạ" để lần sau tự áp dụng cho file cùng định dạng.

Bước 1 — Xác nhận đúng dòng chứa tên cột

Nhiều báo cáo HIS có vài dòng tiêu đề (tên Sở/Bệnh viện, tên báo cáo, khoảng ngày...) phía trên dòng tên cột thật (STT, MÃ KCB...). App đã tự đoán, nhưng anh nên xem lại bảng bên dưới — dòng được tô cam là dòng app đang dùng làm tên cột. Nếu sai, đổi số dòng rồi bấm "Áp dụng".
